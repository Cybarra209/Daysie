# The Great Battle of Daysie

A mobile-friendly tower defense game made with HTML, CSS, and JavaScript.

## How to play locally
1. Open `index.html` in a browser.
2. Pick the male or female stick figure.
3. Earn 1 coin per enemy defeated.
4. Tap turret spots when you have enough coins.
5. Use **Pray** for 10 coins to wipe out enemies and heal your base.

## How to put it on GitHub Pages
1. Create a new GitHub repository.
2. Upload `index.html`, `style.css`, `game.js`, and the `assets` folder.
3. Go to **Settings > Pages**.
4. Under **Build and deployment**, set the source to **Deploy from a branch**.
5. Choose the `main` branch and `/root`, then click **Save**.
6. GitHub will give you a website link after it finishes publishing.
